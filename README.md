## О проекте

Полноценная Tower Defense игра, созданная на Unity.

### Игровой процесс
1. **Враги** идут по заданному маршруту к базе
2. **Строй башни** на специальных платформах, чтобы их остановить
3. **Зарабатывай деньги** за убийство врагов
4. **Покупай улучшенные башни** — ракетные и лазерные
5. **Не дай врагам уничтожить базу** (5 HP)

### Типы башен

| Башня | Стоимость | Особенность |
|-------|-----------|-------------|
| **Стандартная** | 15 | Стреляет пулями по одному врагу |
| **Ракетная** | 135 | Наносит урон в радиусе 8 единиц |
| **Лазерная** | 75 | Замедляет врага до 2 единиц скорости |

### Управление
- **ЛКМ** по платформе — построить башню
- **WASD** или движение мыши к краю экрана — панорамирование камеры
- **Колесо мыши** — зум
- **Escape** — включить/выключить управление камерой

## Технологии
- Unity (версия 2020.3 LTS или новее)
- C# 8.0
- Корутины (спавн волн)
- LineRenderer (лазер)
- Particle System (взрывы)

## Как запустить

Переместите файлы из src на уровень выше.
